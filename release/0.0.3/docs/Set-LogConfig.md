# Set-LogConfig

## SYNOPSIS
Sets the configuration of NLog based on a config file

## SYNTAX

```
Set-LogConfig [-Config] <String>
```

## DESCRIPTION
NLog config file can be customized based at will

## EXAMPLES

### -------------------------- EXEMPLE 1 --------------------------
```
Set-LogConfig -Config ./nlog.config
```

## PARAMETERS

### -Config
NLog Config File Path.
The path must exist.
